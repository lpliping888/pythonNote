## 输入
### 1.input()函数
>input()函数从用户那里得到一个字符串。

* 第1种用法
```python
print("请输入你的名字：")
name=input()
print(name+"，你长的可帅！")
```
>运行结果：<br>
请输入你的名字：<br>
猪八戒<br>
猪八戒，你长的可帅！<br>

* 第2种用法-打印内容不要print函数
```python
name=input("请输入你的名字：")
print(name+"，你长的可帅！")
```
>运行结果：<br>
请输入你的名字:齐天大圣：<br>
齐天大圣，你长的可帅！<br>
