## 循环结构（一）
### 1.什么是循环结构
>在程序设计中，某些操作步骤在一定条件下被重复执行: <br>
@$@$@$@$@$@$@$@$@$@$@$@$@$@$@$@$@$@$@$@$@$@$@$@$@$@$@$@$@$@$@$@$@$@$@$@$@$@$@$@$@$@$@$@$@$@$@$@$@$@$


### 2.基本格式
```
while 条件：
    条件满足时不断的做某件事情
```

### 3.案例
```
i=1
while i<=50:
	print("@",end="")
	print("$",end="")
	i+=1
```
